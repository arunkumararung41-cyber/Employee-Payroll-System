# 💼 Employee Payroll System (Java OOP)

## 📌 Overview

This project is a simple **Employee Payroll System** implemented using **Object-Oriented Programming (OOP)** concepts in Java. It demonstrates abstraction, inheritance, polymorphism, and encapsulation.

The system allows you to:

* Add employees (Full-Time & Part-Time)
* Remove employees
* Display employee details with calculated salary

---

## 🚀 Features

* 🔹 Abstract `Employee` class
* 🔹 Two types of employees:

  * Full-Time Employee
  * Part-Time Employee
* 🔹 Dynamic salary calculation using method overriding
* 🔹 Employee management using `ArrayList`
* 🔹 Clean and modular OOP design

---

## 🏗️ Project Structure

```
Employeeoopsproject/
│
├── Employee (Abstract Class)
├── FullTimeEmployee (Subclass)
├── PartTimeEmployee (Subclass)
├── PayRollSystem (Manager Class)
└── Main (Driver Class)
```

---

## 🧠 OOP Concepts Used

### 🔸 Abstraction

* Abstract class `Employee` with abstract method `CalculateSalary()`

### 🔸 Encapsulation

* Private fields (`name`, `id`) with getter methods

### 🔸 Inheritance

* `FullTimeEmployee` and `PartTimeEmployee` extend `Employee`

### 🔸 Polymorphism

* Method overriding of `CalculateSalary()`

---

## ⚙️ How It Works

1. Create a payroll system
2. Add employees (full-time or part-time)
3. Display all employees
4. Remove an employee by ID
5. Display updated list
